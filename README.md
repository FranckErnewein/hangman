# Hangman

```
 _________
 | /     |
 |/      |
 |       O
 |      /|\
 |      / \
 |
_|_________
```

## Run

assuming node 10 and yarn installed

```
yarn
yarn start
```

then visit [http://localhost:3000](http://localhost:3000)
